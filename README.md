## 개발 기획

1. 제작의도, 서비스 목표

   - 영화 목록을 깔끔한 UI로 제작
   - 친근한 요즘 세대 느낌

2. 클라이언트 요구분석

   - 깔끔하고 보기 쉬운 디자인
   - 원하는 장르의 영화 정리
   - 예고편과 줄거리에 관한 영화 유튜브 링크를 같이 기재

3. 사이트맵 구현

   - 일러스트로 정리

4. 스택(stack)

   - html, css, js, node js, react

5. 디자인
   - 메인, 서브 폰트 및 색상 인터렉션
   - 메인페이지 디자인
   - 서브(디테일, 404, 검색, 로그인 등) 디자인
